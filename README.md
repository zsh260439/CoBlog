# CoBlog

一个探索代码之美与设计之心的博客空间。

## 技术栈

- **前端**: Vue 3 + TypeScript + Vite
- **后端**: Node.js
- **样式**: UnoCSS + 自定义 CSS 变量

## 开始使用

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev
```

## 项目结构

```
├── client/          # 前端应用
│   ├── src/
│   │   ├── components/  # 组件
│   │   ├── views/       # 页面视图
│   │   └── ...
│   └── ...
├── server/          # 后端服务
├── shared/          # 共享类型/配置
└── ...
```
